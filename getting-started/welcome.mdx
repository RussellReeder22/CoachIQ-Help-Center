## Common Setup Questions

<AccordionGroup>
  <Accordion title="Do I need a business bank account?">
    Yes, Stripe requires a business bank account for payouts. If you don't have one yet, you can start the setup process and add bank details later. Most banks can set up a business account in 1-2 days.
  </Accordion>
  <Accordion title="Can I import my existing clients?">
    Absolutely\! After initial setup, you can bulk import clients from a spreadsheet. See [Import Athletes from Spreadsheet](/how-to/import-athletes-spreadsheet) for details.
  </Accordion>
  <Accordion title="What if I don't have a website domain?">
    No problem\! CoachIQ provides a free subdomain (yourbusiness.coachiq.io) to start. You can always add a custom domain later. See [Domain Setup](/website/domain-setup) for options.
  </Accordion>
  <Accordion title="Can I test everything before going live?">
    Yes\! You can create test bookings and preview everything before launching. We recommend doing a test booking with yourself before inviting athletes.
  </Accordion>
</AccordionGroup>
